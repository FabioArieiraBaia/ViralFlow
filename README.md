# ViralFlow AI

**ViralFlow AI** is an automated viral video generator powered by Google's **Gemini 2.5** models. It orchestrates Script Generation, Text-to-Speech (TTS), Image Generation, and Video Rendering into a seamless workflow, running locally in your browser or via Electron.

## üöÄ Features

*   **AI Scriptwriting:** Generates engaging scripts based on topics, styles, and pacing using Gemini 2.5 Flash.
*   **Cinematic TTS:** Uses Gemini 2.5 Flash Native Audio for realistic multi-speaker voiceovers.
*   **Visual Generation:**
    *   Gemini 2.5 Flash Image
    *   Pollinations.ai (Flux/SD)
    *   Pexels Stock Video Integration
*   **Video Rendering:** Real-time canvas rendering with particle effects, overlays, transitions, and subtitles (Karaoke, Neon, etc.).
*   **Local & Secure:** All API keys are stored locally.
